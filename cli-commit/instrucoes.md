# Git Commit History Generator

🌱 Ferramenta para criação automatizada de histórico de contribuições no GitHub

## Sobre

Este repositório contém um script Python que permite gerar automaticamente um histórico de commits retroativos para preencher seu gráfico de contribuições do GitHub. Com ele, é possível criar até 3500 commits datados no passado com apenas uma execução.

## Características

✨ Geração de commits com datas retroativas customizáveis  
📊 Preenche o gráfico de contribuições do GitHub  
🔄 Processo totalmente automatizado  
⚙️ Configuração simples e fácil de usar

## Por que usar?

- Restaurar histórico de contribuições perdidas
- Demonstrar consistência para portfólios
- Testar funcionalidades do GitHub relacionadas ao histórico de commits
- Experimentar com a visualização de dados do gráfico de contribuições

## Como começar

Veja as instruções detalhadas no arquivo README.md para configurar e executar o script em seu próprio repositório.
