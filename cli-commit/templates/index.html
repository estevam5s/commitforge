CommitForge v2.0

CommitForge é uma ferramenta para criação automatizada de commits retroativos no GitHub, GitLab e Bitbucket. Permite preencher o histórico de contribuições com datas do passado.

Como funciona?

O CommitForge clona seu repositório, cria commits com datas retroativas usando as variáveis GIT_AUTHOR_DATE e GIT_COMMITTER_DATE do Git e, opcionalmente, os envia para o repositório remoto via push forçado.

Modos de Data

  • Por Ano: preenche um ano inteiro (ex: 2020)
  • Intervalo: datas específicas de início e fim
  • Últimos N Dias: retroativos a partir de hoje

Recursos

  • Até 5000 commits por processo
  • Múltiplos commits por dia (até 10)
  • Horários aleatórios para look natural
  • Opção de pular fins de semana
  • Personalização de mensagem e autor
  • Monitoramento em tempo real do progresso
  • Autenticação via token do GitHub
  • Interface Web + CLI no terminal

CLI no Terminal

pip install commitforge
commitforge commit --repo https://github.com/user/repo.git --year 2020
commitforge status
commitforge historico

Uso Responsável

Esta ferramenta foi criada para fins legítimos como migração de repositórios, testes de workflow Git e demonstrações. Use com responsabilidade e em conformidade com as políticas da plataforma que você utiliza.
